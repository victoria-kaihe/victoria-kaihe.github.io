What is grid-based design?

So - before we talked about everything being in a box. Gridge-based design is when all those boxes are arranged in a way that makes sense to read. We start at the top, then track from left to right and down.

What's all the hype about responsive web pages?

Responsive design means our web pages can adjust to best fit the size of the screen they're being displayed on - because sometimes, one size just doesn't fit all.

What is semantic structure?

Semantics is about meaning. In html, semantic structure is used to give elements meaning that can be understood by both people, and the the browser. Neat, eh?
