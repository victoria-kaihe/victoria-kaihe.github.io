HTML refers to your most basic content and structure - text, images, links, tables etc. CSS is how those things are styled. Javascript is pretty much all the interactive stuff.

First off, I'd like to apologise for my real life example - it's the first thing that I thought of, and I really would explain it this way to a friend so I'm sticking with it. Control flow refers to the order in which things happen. Loops are a series of events that only need to be specified once but can be repeated any given number of times, indefinitely, or until a condition is met. For example, when you go to the toilet, you take your pants off, then do your business, then you wipe. Wiping is looped until the condition of being clean is met, then wiping stops.

To my understanding, the difference between accessing data in arrays versus objects, is that arrays have a zero-based index (so the first item will be 0, the second will be 1 and so on and so forth), and data in an object are assigned arbitrary keys.

A function is a piece of code that will perform a particular task or calculation when it is invoked. What's useful about them is that they can be reused - write the code once and then feed it different arguments to get different results. An example of this in use would be when building a conversion calculator e.g. kgs to lbs, flOz to mls, ft to m.
