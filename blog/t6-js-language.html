Hello, non-tech friend. I hope some of what follows will make some sense to you.

In JavaScript, parentheses do a couple of different things. They are used for executing functions like...

var result = doSomething()

Parentheses are also used for surrounding conditional statements like...

if (specifiedCondition) {then do this}

And they're used to enforce order of operations, like we all learned at primary school - BEDMAS, BODMAS, BOMDAS...

Brackets [] are generally used for accessing the properties of an object. For example myObject[2] would get the third element in myObject.

Braces {} declare object literals, and enclose blocks of code such as loops.

In JavaScript, ' and " are pretty much the same thing. They're both used to outline a string, but if you're going to use quotes inside a string, it's best to use one kind to enclose the whole string, and the other kind inside it.
